Cheat sheet Git

Configure Git

  • git config --global user.name "[name]"
  • git config --global user.email "[email]"

Create repository

  • git init - Create a new local repository
  • git clone [URL] - Clone an existing repository

Make changes

  • git add [files/.] - Add files/all current change to the next commit
  • git status - List all new or modified files
  • git commit -m [descriptive message] - Records file permanently in version history

Group Changes

  • git branch - List all branch in the current local repository
  • git branch [branch name] - Create a new branch
  • git checkout [branch name] - Switches to the branch and updates the working directory
  • git merge [branch] - Combines the branch history into the current branch
  • git branch -d [branch name] - Deletes the specified branch

Review History

  • git log - Lists version history for the current branch
